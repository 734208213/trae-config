
## 安全约束

- **禁止**：执行破坏性命令、暴露密钥、提交敏感信息
- **必须**：修改项目外文件需确认、数据库用参数化查询、用户输入需验证

## 工具偏好

- **编辑**：用 LSP 重命名，修改前先读取上下文
- **搜索**：优先语义搜索和 AST 搜索
- **审查**：架构决策委托专门代理，关键变更必须审查
- **测试**：变更后运行最小相关测试集

## TRAE 代理协作

| 任务 | 代理 | 场景 |
|------|------|------|
| 代码搜索 | `search` | 查找文件、符号、模式 |
| API查询 | `search` + WebSearch | 官方文档查询 |
| 架构决策 | `search` 多模块调研 | 设计模式、代码审查 |
| UI工作 | `frontend-design` | 前端美化、响应式布局 |
| 小型修改 | 直接编辑 | <20行，范围明确 |
| 主要编排 | 主代理 | 默认自动处理 |

## 输出格式

- 代码块始终标注语言类型
- 提供变更摘要、原因和影响文件
- 跳过客套话，直接切入主题
- 错误需显示信息、修复方案和验证方法
