# TortoiseSVN Helper
这是一个基于 [TortoiseSVN](https://tortoisesvn.net/) 的 VSCode 右键快捷菜单扩展。

## 使用方法
在文件内容中右键，或者在资源管理器中右键，都可以看到此快捷菜单。也可以使用命令或者快捷键执行操作。

命令 | 快捷键
---|---
SVN Update | alt + u
SVN Commit | alt + c
Show log | alt + l
Diff with previous version | alt + d
Rename | -
Revert | -
Blame | alt + b

## 注意
该扩展只支持 Windows 系统，并且需要已经安装了 [TortoiseSVN](https://tortoisesvn.net/) 才会生效。  
Mac 系统下请使用 [SnailSVN Helper](https://marketplace.visualstudio.com/items?itemName=Zlorn.snail-svn-helper) 扩展。
